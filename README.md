Данный репозиторий был создан с целью командной работы по разработке веб-приложения по предмету ЖЦРПО. Участники команды: Никита, Миша, Дима - студенты группы 334701. 
В процессе выполнения репозиторий будет постепенно дополняться новыми файлами агоритмов, документации и других обновлений.
Наш проект представляет собой приложение по генеерации рецептов из имеющихся ингредиентов. Оно подойдет как для обычных пользователей, которые хотят приготовить что-то простенькое и быстрое, так и для 
настоящих гурманов, стремящихся получить гастрономическое удовольствие.
